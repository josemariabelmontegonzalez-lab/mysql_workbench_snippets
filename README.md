# MySQL Workbench Snippets Installer

Herramienta para instalar automáticamente una colección de snippets SQL en MySQL Workbench con un solo clic.

## ¿Qué hace?

Copia el archivo de snippets a la ruta de snippets de MySQL Workbench, haciendo **backup automático** del archivo anterior si ya existe.

## Contenido

- +80 funciones MySQL documentadas con sintaxis y 3 ejemplos cada una
- Categorías cubiertas: cadena, fecha/hora, numéricas, control de flujo, sistema

## Uso

1. Clona o descarga el repositorio
2. Asegúrate de tener **MySQL Workbench** instalado
3. Ejecuta "install.bat" como administrador
4. Abre MySQL Workbench → los snippets estarán disponibles en el panel lateral

## Requisitos

- Windows
- MySQL Workbench instalado
- PowerShell (incluido en Windows por defecto)

## Tecnologías

`PowerShell` `Batch` `MySQL`
